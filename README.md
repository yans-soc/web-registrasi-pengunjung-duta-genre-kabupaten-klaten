# Web Registrasi Pengunjung Duta Genre Kabupaten Klaten

Aplikasi web registrasi dan sistem ticketing untuk pengunjung pemilihan Duta Genre Kabupaten Klaten.

## Fitur Utama

- Registrasi Pengunjung Mandiri
- Sistem Ticketing & QR Code Unik
- Check-in QR Code Scanner untuk Panitia
- Dashboard Admin untuk Manajemen Data
- Pengaturan Custom Section dan Form Fields Dinamis
