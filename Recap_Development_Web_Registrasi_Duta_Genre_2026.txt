WEB REGISTRASI PENGUNJUNG – MALAM PUNCAK DUTA GENRE 2026

Tujuan

Membangun aplikasi web registrasi pengunjung berbasis QR Code tanpa
login untuk pengunjung, dengan dashboard admin yang memiliki dua role
(Super Admin dan Checker).

------------------------------------------------------------------------

TECH STACK

Frontend - Next.js 15 (App Router) - TypeScript - Tailwind CSS -
shadcn/ui

Backend - Next.js Route Handlers (API)

Database - MySQL

ORM - Prisma

Authentication Admin - JWT + HTTP Only Cookie

QR - Generate: qrcode - Scanner: html5-qrcode

------------------------------------------------------------------------

ROLE

Super Admin

Akses: - Dashboard - Data Pengunjung - Scanner QR - Verifikasi Manual -
Statistik - Export Excel - Manajemen Admin - Hak Akses Admin -
Pengaturan - Activity Log

Checker

Akses: - Scanner QR saja

Saat login Checker langsung diarahkan ke: /admin/scanner

------------------------------------------------------------------------

FLOW PENGUNJUNG

1.  Homepage
2.  Klik Daftar
3.  Isi Form:
    -   Nama Lengkap
    -   Jenis Kelamin
    -   Tanggal Lahir
    -   Umur (otomatis dari tanggal lahir)
    -   Alamat Lengkap
4.  Submit
5.  Sistem menyimpan data
6.  Generate QR Code + Unique Code
7.  Tampilkan halaman QR
8.  User download / screenshot QR
9.  Datang ke venue
10. Checker scan QR
11. Status berubah menjadi CHECKED IN

Tanpa login.

------------------------------------------------------------------------

FITUR AMBIL ULANG QR

Menu: “Saya Sudah Daftar”

Input: - Nama Lengkap - Tanggal Lahir

Jika data ditemukan: QR ditampilkan kembali.

------------------------------------------------------------------------

FLOW ADMIN

Login ↓ Cek Role

Jika Super Admin → Dashboard

Jika Checker → Scanner QR

------------------------------------------------------------------------

DASHBOARD SUPER ADMIN

Menu: - Dashboard - Pengunjung - Scanner QR - Laporan - Manajemen
Admin - Pengaturan - Logout

Dashboard menampilkan: - Total Registrasi - Sudah Check In - Belum
Hadir - Checker Aktif

------------------------------------------------------------------------

DASHBOARD CHECKER

Menu: - Scanner QR - Logout

------------------------------------------------------------------------

FORM REGISTRASI

Field: - Nama Lengkap - Jenis Kelamin - Tanggal Lahir - Umur (readonly
otomatis) - Alamat Lengkap

Button: Daftar

------------------------------------------------------------------------

HALAMAN QR

Menampilkan: - Nama - Unique Code - QR Code - Tombol Download QR

QR payload: - visitor_id - unique_code - signature

Gunakan signature agar QR tidak mudah dipalsukan.

------------------------------------------------------------------------

SCANNER

Menggunakan kamera.

Jika valid: - tampilkan nama - tampilkan jam - status CHECK IN - simpan
waktu checkin - otomatis kembali scan

Jika sudah pernah check in: Tampilkan: “QR SUDAH DIGUNAKAN”

------------------------------------------------------------------------

VERIFIKASI MANUAL

Input: Unique Code

Jika valid: Status menjadi CHECKED IN.

------------------------------------------------------------------------

MANAJEMEN ADMIN

Super Admin dapat: - Tambah Admin - Edit Admin - Hapus Admin -
Nonaktifkan Admin - Reset Password - Atur Hak Akses

Role: - superadmin - checker

Walaupun role hanya dua, permission disimpan terpisah agar mudah
dikembangkan.
